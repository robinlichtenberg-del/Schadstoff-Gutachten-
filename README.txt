WICHTIG FÜR NETLIFY / PWABUILDER

1. ZIP entpacken.
2. Den INHALT des Ordners schadstoff_profi_pwabuilder hochladen:
   - index.html
   - manifest.json
   - service-worker.js
   - Ordner icons
3. Nicht die ZIP hochladen.
4. Nicht den übergeordneten Ordner hochladen, sondern die Dateien direkt.
5. Danach die Netlify-Adresse in Chrome öffnen.
6. Dann bei PWABuilder diese Netlify-Adresse eintragen.
